# HotShop
E-Commerce Project for Sporting Equipments with the features of Admin, Seller and Customer
E-commerce Website using React and Redux for Front-end, Java Springboot, Java Mail Sender, with MySQL as Database both connected using Hibernate, Razorpay is used as payment gateway and Frontend is connected to backend using axios and Used Gemini as AI chatbot for customer support services. Website consists of Admin, Seller, Customer panels which provides vast user experience
